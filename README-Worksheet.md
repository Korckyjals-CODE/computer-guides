# Worksheets – For the Teacher

## Purpose

These worksheets are **self-contained** learning activities that teach technology and digital literacy concepts. Each worksheet has instructional value on its own—students learn from the content and activities regardless of any computer-based project.

## Use

- **When:** Use a worksheet whenever the topic fits your curriculum. There is no required “before the project” timing.
- **Format:** Students work **offline**, with **pen and paper** (and color pencils if you like). No computer or internet needed.
- **Time:** About **25 minutes** per worksheet. Each part suggests roughly 3–6 minutes so students can pace themselves.
- **Guidance:** The text is written so that students can complete it **on their own**; no teacher guidance is required during the activity.

## Files

- **`[Topic-Name]-Worksheet.html`** — Open in a browser and **print** (File → Print or Ctrl+P). Give each student a printed copy.
- **`[Topic-Name]-Worksheet.md`** — Source text. You can paste it into Word or Google Docs to adjust wording or layout, then print.

## Learning objectives

Each worksheet states what students will learn. Typical objectives include:

- Key concepts for the topic (e.g. what a computer is, parts of an email, data and charts).
- Skills such as identifying, classifying, planning, or applying concepts.
- Critical thinking through discussion questions, scenarios, or reflection.

## Standalone value

Worksheets teach on their own. No computer project is required. If a TBox (or other) computer project exists for the topic, it can be used as an optional follow-up for hands-on practice, but the worksheet does not depend on it.

## Folder structure

- **samples/** — Example worksheets by grade (e.g. `1-grade/`, `3-grade/`, `5-grade/`).
- **worksheets/** — Generated worksheets organized by grade and week.
