# Problems Technology Solves – Worksheets (Grade 6, Week 38)

## For the teacher

- **Use:** Use these worksheets **whenever** the topic of how technology solves real-world problems fits your curriculum. They are self-contained learning activities, not tied to a specific computer project.
- **Format:** Students work **offline**, with **pen and paper** (and color pencils if you like). No computer or internet needed.
- **Time:** About **25 minutes** per worksheet. Each part has a suggested time (roughly 3–6 minutes) so students can pace themselves.
- **Guidance:** The text is written so that students can complete it **on their own**; no teacher guidance is required during the activity.
- **Standalone value:** Each worksheet teaches on its own. The two sheets are **independent**—either can be used first in the same week. No computer project is required.

## Learning objectives

### Technical worksheet

- Explain that technology is designed to address **problems** (a gap between how things are and how we want them).
- Name common **problem types** technology tackles (repetitive work, distance, information overload, safety, communication).
- Match everyday **problems** to **tools** that help solve them.
- Describe **input → process → output** and optional **feedback** in simple systems.
- Use a basic **solve-it process**: define the problem, list options, pick and test, improve—and notice **constraints** and **tradeoffs**.

### Social worksheet

- Ask **whose problem** gets solved when technology is introduced, and who may be left out.
- Describe **access** gaps (devices, connectivity, language, accessibility) and fairness when technology is required.
- Recognize that technology can solve one problem and create **new concerns** (privacy, pressure, distraction).
- Use questions to decide when technology is the **right** choice (need, simpler fix, affected voices).
- Practice **respectful communication** and naming a **trusted adult** when a “solution” feels unfair.
