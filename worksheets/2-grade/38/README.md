# How I Use Technology – Worksheets (Grade 2, Week 38)

## For the teacher

- **Use:** Use these worksheets **whenever** the topic of how students use technology fits your curriculum. They are self-contained learning activities, not tied to a specific computer project.
- **Format:** Students work **offline**, with **pen and paper** (and color pencils if you like). No computer or internet needed.
- **Time:** About **25 minutes** per worksheet. Each part has a suggested time (roughly 3–5 minutes) so students can pace themselves.
- **Guidance:** The text is written so that students can complete it **on their own**; no teacher guidance is required during the activity.
- **Standalone value:** Each worksheet teaches on its own. No computer project is required; optional hands-on activities may complement them if you choose.

## Learning objectives

### Technical worksheet

- Define **technology** as tools that use electricity or batteries.
- Name **where** and **when** students use technology in daily life.
- Match tools to jobs such as **watch**, **listen**, **learn**, **play**, and **talk**.
- Choose the **right tool for the job** and reflect on what happens when the wrong tool is used.

### Social worksheet

- Follow **permission** and **time limits** when using others’ devices.
- Take **fair turns** and use **kind words** when sharing technology.
- Practice **balance** and **healthy breaks** for eyes and body.
- Tell a **trusted adult** when something online feels wrong and protect **privacy** (do not share full name, address, or photos with strangers).
